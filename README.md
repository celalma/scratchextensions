# Extensions

Learn how to create your own sandoboxed and unsandboxed Scratch/Turbowarp/Penguinmod extensions for your own projects (games, applications,...) whithout having to learn too much JavaScript.
Just imagine having your own blcoks that you can use just like Scratch blocks !

## Why would i need to create extension?

Creating extensions is a good idea for a lot of reasons :
- Regroup functionnalities that are specific or that you use a lot in fewer blocks
- Get acces to functionnalities that aren't in the base Scratch
- And much more...


Although it is quite cool and very useful, if comes with a downside :
### You cannot use these extensions directly in Scratch and you cannot upload and post projects that use custom extenions.

## So, how do I use these custom extensions?

To use custom extensions, you need to use a third-party editor such as Turbowarp or Penguinmod. Note that while you cannot share projects on Turbowarp, you can convert them to HTML or Electron using the [Turbowarp packager](https://packager.turbowarp.org) and you can share them on Penguimod.

Turbowarp and Penguimod already have some more custom extensions. If you head to the **Extension tab** (bottom left hand corener of the editor), you canbrowse custom extensions made by the community. If you search for **"Custom extensions"**, and click on it, you will see that you can import Javascript files by either copy-pasting / writing directly, uploading a file or loading it through a direct URL to the extension.
